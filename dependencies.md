## Crear un nuevo proyecto con GraphQL y Apollo Server

```console
$ npm init
$ npm i apollo-server
$ npm i -D nodemon
```
#### Configurar scripts en package.json
```console
"scripts": {
    "start": "node .",
    "dev": "nodemon ."
  },

$ npm run dev
```
#### Crear el servidor de Apollo Server
Crear el archivo *index.js*


#### Instalar Mongoose
```console
$ npm i mongoose dotenv
```

#### Hashear el password
```console
$ npm i bcryptjs
```

#### Crear un token con JWT
```console
$ npm i jsonwebtoken
```
